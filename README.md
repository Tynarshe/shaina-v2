# Shaina v2

Static multi-page redesign for Shaina Cleaning Services.

## Pages

- `index.html`
- `about.html`
- `services.html`
- `faq.html`
- `job-application.html`
- `contact.html`

## Local Preview

Run a static server from this folder:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## Notes

- The logo asset is included locally at `assets/img/shaina-logo.png` and is based on the Shaina logo referenced by the source site at `https://shaina.org.uk/`.
- Forms validate in-browser and open a pre-filled email to `enquiries@shaina.org.uk`.
- No build step or package install is required.
